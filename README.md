# light-kit.js
This is a JavaScript library that makes coding easier and it made by TypeScript. This is also a frontend library. This library is makes for simple code. The JavaScript function are very simple at there.

## Features
The unique part is type safety and no compilation.

1. Math functions 
2. String functions
3. Array functions

## Usage
```
// import math functions from light-kit js 
import { lightMath } from "light-kitjs";

// the arguments are type, minimum and maximum number.
const num = lightMath.randNum("int", 1, 15);

console.log(num);

```

## Installation
```
npm i light-kitjs
```
